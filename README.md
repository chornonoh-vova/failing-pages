# failing-pages

> [!IMPORTANT]  
> Requires Node.js v24+

Installing dependencies:

```bash
npm install
```

Run with:

```bash
npm start
```

## Available routes:

- `/pages` - lists all available testing pages
  - `/pages/:page` - specific page
- `/http` - lists all available http status pages
  - `/http/:status` - responds with particular http status
